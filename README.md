# ProgramacaoHN 🌐

## Descrição

ProgramacaoHN é um projeto educacional desenvolvido para demonstrar conceitos fundamentais de desenvolvimento web front-end. O projeto apresenta uma aplicação web interativa que combina HTML semântico, CSS responsivo e JavaScript dinâmico, focando especialmente na implementação de funções e manipulação do DOM.

## 🚀 Funcionalidades

- **Interação com Usuário**: Coleta de nome através de prompt nativo
- **Manipulação DOM**: Atualização dinâmica de conteúdo da página
- **Efeitos Visuais**: Alternância automática de cores RGB em tempo real
- **Design Responsivo**: Interface adaptável para dispositivos móveis
- **Interface Intuitiva**: Botões interativos para controle de funcionalidades

## 🛠️ Tecnologias Utilizadas

- **HTML5**: Estrutura semântica e acessível
- **CSS3**: Estilização com design responsivo e media queries
- **JavaScript (ES6+)**: Lógica de interação e manipulação DOM
- **Git**: Controle de versão

## 📋 Pré-requisitos

- Navegador web moderno (Chrome, Firefox, Safari, Edge)
- Servidor local (opcional, para desenvolvimento)

## ⚡ Como Executar

### Opção 1: Execução Direta
1. Clone o repositório:
```bash
git clone https://github.com/willianbobelapr/program-oHN.git
```

2. Navegue até o diretório:
```bash
cd program-oHN
```

3. Abra o arquivo `index.html` em seu navegador

### Opção 2: Servidor Local
1. Com Python 3:
```bash
python -m http.server 8000
```

2. Acesse: `http://localhost:8000`

## 🎯 Objetivos de Aprendizagem

Este projeto foi desenvolvido para ensinar:

1. **Conceitos de Função**: Compreensão e implementação de funções JavaScript
2. **Integração HTML-JS**: Comunicação entre marcação e lógica
3. **Manipulação DOM**: Seleção e modificação de elementos da página
4. **Event Handling**: Gerenciamento de eventos e interações
5. **Design Responsivo**: Adaptação para diferentes dispositivos

## 🔧 Funcionalidades Técnicas

### JavaScript
- Função `inserirNome()`: Coleta input do usuário e atualiza DOM
- Função `alternarCor()`: Implementa efeito visual RGB automático
- Event Listeners: Gerenciamento de eventos de carregamento
- setInterval: Execução de código em intervalos regulares

### CSS
- Design responsivo com media queries
- Pseudo-classes para interatividade (hover)
- Layout flexível e centralizado
- Tipografia otimizada para legibilidade

## 🎨 Demonstração

1. **Carregamento**: Página solicita nome do usuário automaticamente
2. **Personalização**: Nome substitui placeholder padrão
3. **Efeito Visual**: Alternância automática entre cores RGB (1s)
4. **Interação**: Botão para alteração posterior do nome

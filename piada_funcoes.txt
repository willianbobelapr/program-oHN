🤣 PIADA DAS FUNÇÕES COM CÓDIGO JS 🤣

Por que a função estava triste?

function funcaoTriste() {
    // Ela sempre retornava undefined! 😢
}

console.log(funcaoTriste()); // undefined


Mas aí ela conheceu uma função callback...

function funcaoFeliz(callback) {
    console.log("Executando junto com minha callback! ❤️");
    callback();
}

function meuCallback() {
    console.log("Agora vivemos executando juntas!");
}

funcaoFeliz(meuCallback); // E agora elas vivem executando juntas! ❤️

Por que as funções JavaScript nunca brigam?

function funcao1(parametro) {
    console.log("Oi " + parametro + "! 😄");
}

    console.log("Oi " + parametro + "! 😄");
}

// Porque elas sempre se chamam pelos parâmetros corretos! 😄
funcao1("JavaScript");
funcao2("Programação");
